HTML é a linguagem de marcação usada para fazer a estrutura de um website, agindo geralmente em conjunto com o CSS para estilizar o conteúdo. Com o uso de tags, o conteúdo é declarado de maneira explícita através dessas tags que formam a estrutura do site.

- Principais tags
    
<h1> a <h6>: Usada para definir títulos e subtítulos, formando uma hierarquia para esses.

<p>: Usada para definir parágrafos.

<br>: Adiciona uma quebra de linha na página e uma fina linha horizontal

<img>: Adiciona uma imagem no site.

<a>: Representa links em um site.

<ul> e <ol>: Representa lista desordenada e ordenada consecutivamente.

<li>: Representa um item da lista.

<div>: Muito usada para agrupar outras tags. Facilitando assim a criação de interfaces e a compreensão do código.

<input>: Usada para receber dados digitados por um usuário.

<button>: Representa botões clicáveis.

<body>: Representa toda a página web.

<head>: Representa metadados da página.

- Tags semânticas

<nav>: Representa uma narra de navegação pelo site.

<main>: Representa o principal conteúdo da página.

<aside>: Representa uma sidebar em um site.

<article>: Representa um elemento auto-contido, ou seja, que seja entendível por si só.

<section>: Representa uma seção em uma página.

<footer>:Representa um rodapé em um site.
