# Projeto NutriAI

Trata-se de um projeto desenvolvido com o objetivo de solucionar problemas relacionados ao gerenciamento de receitas culinárias, ao mesmo tempo em que permite o rastreamento da quantidade de calorias consumidas pelo usuário em seu dia a dia. A aplicação visa oferecer maior controle alimentar, organização e praticidade, auxiliando usuários a adotarem hábitos mais saudáveis por meio do acompanhamento contínuo de suas refeições.
